INCONTOURNABLES

Les extensions Vscode:

EMMET

Permet de générer du code HTML et CSS avec seulement des abréviations

Il faut l'activer pour les fichiers Html.erb en incluant "emmet.includeLanguages": {"erb": "html"} dans les paramètres de langage d'Emmet.

Endwise

Ajoute automatiquement la balise 'end' à la fin des methodes ou conditions en Ruby.

ERB-VSCode-Snippets

Permet d'écrire des balises entières avec des abréviations.

Exemples d'abréviations

VS Code Icons

Change les icônes d'origines par des icônes qui correspondent aux fonctions du dossier ou fichier.


Ruby

Cette extension fournit une prise en charge améliorée du langage Ruby et du débogage pour Visual Studio Code.


Raccourcis clavier vscode Linux

CTRL+SHIFT+F

Ouvre le menu recherche.

CTRL+H

Remplace toutes les occurrences mise en sur brillance.

CTRL+P

Pour ouvrir rapidement des fichiers.

ALT+⬆️ ou ⬇️

Permet de déplacer la partie sélectionnée.

ALT+ click droit

Positionne plusieurs curseurs où tu veux.

CTRL+SHIFT+⬆️ ou ⬇️

Positionne plusieurs curseurs à la suite.

CTRL+L

Sélectionne la ligne où se trouve le curseur.

CTRL+D puis ⬇️

Permet de dupliquer la ligne.
(Pour linux le rajouter au préferences raccourcie clavier)
